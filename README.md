# Word Search

A program that searches for words.

## Setup

1. Clone the repository: 

	`git clone https://github.com/eudai/word-search.git`

2. Install local dependecies: 

	`npm install`

3. Install global dependencies: 

	`npm i -g jasmine`


## Testing

1. Run the tests:
	
	`npm test`
